优化方向记录：

1. 关注每个new 如果可以单例的话，没有必要每个连接都要new一个对象，使用单例模式

- *注意：千万记得要加 `@ChannelHandler.Sharable` 注解*

2. 有一些参数写死了的，static final的，探讨是否存在延迟产生的可能

3. 调用 `ctx.xxx()` 可以直接跳到编译码处理器， 替换`ctx.channel().xxx()`
